---
title: fragments + inject
description: Reusable step blocks spliced in with inject.
---

Define reusable step blocks at the top of the file; splice them in with
`- inject: <name>`.

```yaml title=".actio.yml"
name: Fragments
on: [push]
fragments:
  setup:
    - uses: actions/checkout@v4
    - uses: actions/setup-node@v4
      with:
        node-version: 20
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - inject: setup
      - run: npm test
```

`fragments` is stripped, `inject` expanded in place:

```yaml title="generated .yml"
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
      - run: npm test
```
