---
title: Installation
description: Install the Actio CLI.
---

```package-install
npm install -D actio-cli
```

Or run it once without installing:

```bash
npx actio-cli build
```

<Callout title="Requirements">
  Actio requires **Node ≥ 20**.
</Callout>

Once installed, head to the [Quickstart](/docs/quickstart) to scaffold and compile
your first source file.
